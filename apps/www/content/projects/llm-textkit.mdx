一个基于大语言模型的智能文本分析平台，集成情感分析与标注、文本聚类分析和智能语境翻译三大核心功能。利用 LLM 的语义理解能力，提供从数据上传、分析处理到结果下载的完整工作流程，支持批量处理和多种 AI 服务集成。

## 适用场景

用户反馈分析、社交媒体监控、客户评价分析、员工调研分析、内容分类整理、多语言数据分析、跨语言文档翻译、舆情分析。

## 核心功能

### 1. 情感分析与标注

![情感分析界面](/projects/llm-textkit/sentiment-analysis.png)

基于大语言模型的文本内容分析工具，支持三个维度的智能判断：

- **有效性判断**：自动识别无效或低质量的文本内容
- **情感分类**：将文本分类为正向、中性、负向三种情感倾向
- **敏感信息检测**：检测文本是否包含敏感信息

支持直接输入单条文本或上传 CSV 文件批量分析。系统会根据提供的文本主题进行语境化分析，确保判断的准确性。

![批量分析进度](/projects/llm-textkit/analysis-progress.png)

批量处理时会显示实时进度条，分析完成后展示详细结果表格，包含有效性、情感类别和敏感信息标注。所有结果支持一键导出为 CSV 文件。

### 2. 文本聚类分析

![文本聚类分析](/projects/llm-textkit/text-clustering.png)

利用大语言模型的语义理解能力，自动识别并分类大量文本中的主要话题：

- **自动聚类**：系统自动发现话题并生成分类
- **自定义分类**：支持用户自定义分类类别和描述
- **单标签/多标签**：灵活选择单标签或多标签分类方式
- **参数调优**：支持调整类别数量和聚类处理大小

分类方式包括单标签签分类（每条文本只归属一个类别）和多标签分类（每条文本可归属多个类别）。自动聚类模式下，系统会智能生成类别名称和描述，用户可以在分类前审核和编辑这些类别。

![聚类结果展示](/projects/llm-textkit/clustering-results.png)

聚类完成后，展示每条文本的分类结果。用户可以审核自动生成的类别定义，确认无误后开始批量分类。分类结果表格展示文本内容及其对应的类别，支持下载为 CSV 格式。

### 3. 智能语境翻译

![智能语境翻译](/projects/llm-textkit/translation.png)

基于上下文理解的高质量多语言翻译工具：

- **语境理解**：根据文本主题进行更准确的翻译
- **批量处理**：支持 CSV 文件批量翻译
- **智能检测**：自动检测并跳过中文文本
- **异步处理**：并发翻译提升处理效率

输入文本主题后，系统会基于主题进行语境化翻译，相比通用翻译工具能提供更符合专业领域的译文。支持直接输入单条文本或上传 CSV 文件批量翻译。

## 技术特性

**异步批量处理**  
采用 asyncio 和 aiohttp 实现异步并发处理，大幅提升批量文本的处理效率。支持自定义并发数量和批次大小，平衡处理速度与 API 成本。

**LLM 调用监控**  
集成 Langfuse 监控（可选），实时追踪所有 LLM 调用的 Token 消耗、响应时间和调用链路。帮助优化 Prompt 设计和控制 API 成本。

**本地缓存机制**  
实现 LLM 调用缓存，相同输入自动复用历史结果，减少重复调用和 API 开销。缓存数据存储在本地，保护数据隐私。
