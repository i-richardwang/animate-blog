一个基于 LangGraph 的智能查询机器人，将自然语言业务问题转换为 SQL 查询。采用 13 个专业化处理节点构成的图状态机架构，支持意图分析、关键词提取、术语映射、查询重写、数据源识别等智能处理流程。

集成向量语义搜索提升查询准确性，支持 MySQL/PostgreSQL 多数据库，具备完整的错误恢复和 SQL 修复机制。
