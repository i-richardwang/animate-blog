一个专为理解文本向量化技术打造的交互式演示平台，通过直观的可视化和实际操作，让开发者轻松掌握 Embedding 技术的核心概念。涵盖从基础概念、相似度计算、语义聚类到向量算术和智能检索的完整演示路径，结合 RAG 等实际应用场景，帮助快速理解并应用这项 AI 时代的核心技术。

## 演示路径设计

Embedding Playground 采用渐进式演示设计，将复杂的向量化概念拆解为五个相互关联的演示模块。从最基础的"什么是向量化"开始，逐步深入到几何直觉、语义理解、数学特性，最终展示在 RAG 等实际场景中的应用。

每个模块都结合了理论解释、交互式操作和可视化展示，让抽象的数学概念变得具象可感。通过动手实践和观察现象，帮助建立对 Embedding 技术的深入理解。

## 核心功能

### 1. 基础概念入门

![基础概念演示](/projects/embedding-playground/basic-concept.png)

快速体验文本向量化的核心功能。输入任意两个词汇，系统会实时计算它们的语义相似度。界面展示相似度评分标准、向量维度等关键信息，并提供友好的概念解释。

通过预设的示例组合（国王 vs 女王、汽车 vs 卡车、苹果 vs 键盘），快速建立对相似度计算的直觉认知。右侧技术参数面板展示使用的 Embedding 模型和处理流程，帮助理解从文本到向量再到相似度的完整转换过程。

### 2. 相似度几何直觉

![相似度几何演示](/projects/embedding-playground/similarity-geometry.png)

将抽象的相似度计算转化为可见的几何关系。通过交互式滑块调整两个向量的坐标，实时观察余弦相似度和夹角的变化。

可视化面板展示两个向量在二维空间中的位置关系，配合实时计算的相似度数值和夹角度数。提供"相似向量"、"不相似向量"、"垂直向量"等预设示例，帮助理解夹角与相似度的关系：夹角越小相似度越高，垂直时相似度接近 0，反向时为负值。

核心洞察：向量间的角度决定相似程度，相似度本质上是在测量方向的一致性而非距离。

### 3. 语义聚类可视化

![语义聚类分析](/projects/embedding-playground/semantic-clustering.png)

观察语义相近的词汇在向量空间中自然聚集的现象。系统使用 PCA 降维技术将高维向量（1024 维）投影到二维或三维空间，让语义关系变得可视化。

左侧提供基础分类（推荐）或自定义词汇输入。可以查看系统如何通过向量化将"就像把朋友们放在房间里，志趣相投的人会自动走到一起"——向量化技术让计算机也能"看到"这种语义关联。

可视化图表中，相同颜色的点代表同一语义类别。通过观察点的分布，可以直观理解：意思相近的词在向量空间中确实会距离更近，形成自然的聚类。这种特性是推荐系统、文档分类等应用的基础。

### 4. 向量算术探索

![向量算术演示](/projects/embedding-playground/vector-arithmetic.png)

展示向量空间中的数学运算特性，经典案例"国王 - 男人 + 女人 ≈ 女王"证明了 Embedding 不仅捕捉语义，还能捕捉概念之间的关系。

界面提供预设示例供选择，也支持自定义算术组合实验。选择三个词汇后，系统执行向量计算并返回最接近结果的词汇。右侧向量表示热力图以视觉方式展示相似度模式，不同颜色的条纹代表向量各维度的数值分布。

计算结果展示期望公式的相似度评分，并附带详细解释。通过观察 A+B-C 组合的结果向量与实际词汇的匹配程度，可以理解向量不仅存储词义，更存储了概念间的结构性关系。这种特性使得迁移学习和类比推理成为可能。

### 5. 智能检索应用

![智能检索演示](/projects/embedding-playground/rag-retrieval.png)

基于真实 HR 知识库展示 RAG（检索增强生成）的实际应用场景。用户输入自然语言问题，系统通过向量相似度搜索找到最相关的文档内容，展示 Top 3 匹配结果。

每个结果卡片显示文档标题、相似度评分和详细内容。与传统关键词搜索不同，语义检索能理解问题的意图。例如询问"我有多少天年假"，系统会匹配到"年假制度"相关文档，即使问题中没有出现"制度"二字。

界面提供多个预设问题（工作时间、福利待遇、培训制度等），也支持自定义输入。对比展示部分帮助理解语义搜索相较关键词匹配的优势：能理解同义表达、语境关系，找到真正相关的内容而非简单的词汇匹配。

这个演示展示了企业知识库、客服系统、文档检索等场景中 RAG 技术的实际价值。

```

├── app.py                 # Streamlit 应用入口

│   ├── intro.py          # 基础概念

│   ├── clustering.py     # 语义聚类
│   ├── arithmetic.py     # 向量算术
│   └── retrieval.py      # RAG 检索
├── utils/                 # 工具模块

│   ├── visualizations.py # 可视化工具
│   └── data.py          # 示例数据
└── pyproject.toml        # 依赖管理（uv）
```
