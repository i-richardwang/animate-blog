一个智能代码提取工具，帮助开发者从任何 GitHub 仓库中快速找到并提取所需代码。通过 AI 驱动的自然语言搜索，提取完整相关文件并自动格式化为适合 Claude、Cursor 等 AI 工具使用的格式。

同时生成清晰的目录树和架构图，帮助快速理解项目结构。支持私有仓库，即将支持 MCP Server 和 REST API 集成。
