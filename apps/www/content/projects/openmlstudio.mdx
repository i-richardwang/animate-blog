一个开源的机器学习建模平台，支持二分类和回归任务。提供从数据上传到模型训练、评估、SHAP 解释和预测的完整工作流程，通过 Web 界面完成机器学习建模。

## 系统架构

![系统架构](/projects/openmlstudio/architecture.png)

OpenMLStudio 采用模块化设计，从用户数据输入开始，通过 Streamlit Web UI 进行交互，核心包含训练模块和预测模块两大引擎。训练模块支持四种算法（Random Forest、Decision Tree、XGBoost、Linear Regression），并集成 SHAP 解释、性能可视化和模型评估等分析工具。底层基于 scikit-learn、XGBoost、SHAP、Plotly 和 Optuna 等成熟技术栈构建。

## 使用流程

### 1. 数据上传与预览

![数据上传与预览](/projects/openmlstudio/data-upload.png)

上传 CSV 或 Excel 文件后，系统自动识别特征类型（数值型、分类型），并展示数据预览。用户可以选择目标变量和用于建模的特征，系统会检测是分类任务还是回归任务。

### 2. 模型选择与训练

![模型选择与训练](/projects/openmlstudio/model-training.png)

选择算法（Random Forest、Decision Tree、XGBoost、Linear Regression），配置训练/测试集划分比例。Random Forest 和 XGBoost 支持 Optuna 自动超参数调优，可以配置调优范围和优化目标。系统还提供数据预处理选项，如标准化和独热编码。

### 3. 模型评估

![模型评估结果](/projects/openmlstudio/model-evaluation.png)

训练完成后，系统展示详细的性能指标。对于分类任务，提供 ROC-AUC、准确率、混淆矩阵和分类报告。对于回归任务，提供 MSE、RMSE、MAE 和 R² 等指标。所有模型训练记录都会被保存，可以查看历史模型的性能对比。

### 4. SHAP 可解释性分析

![SHAP 可解释性分析](/projects/openmlstudio/shap-analysis.png)

基于 SHAP 的可解释性分析。展示全局特征重要性，帮助理解哪些特征对模型预测最重要。SHAP 依赖图展示单个特征与目标变量的关系，以及特征之间的交互效应。

### 5. 批量预测

![批量预测功能](/projects/openmlstudio/batch-prediction.png)

加载已保存的模型，上传新数据进行批量预测。系统会验证数据格式是否与训练时一致，展示预测结果分布和详细的预测值，支持下载预测结果。

## 技术特性

**算法支持**  
Random Forest、Decision Tree、XGBoost、Linear Regression。其中 Random Forest 和 XGBoost 支持基于 Optuna 的贝叶斯优化自动调参。

**可解释性分析**  
集成 SHAP 分析，提供全局特征重要性、单样本解释和特征依赖可视化。帮助理解模型的决策逻辑。

**模型管理**  
自动保存所有训练记录，支持模型持久化和版本管理。可随时加载历史模型进行预测或对比分析。

**数据处理**  
自动识别特征类型，支持标准化、独热编码等预处理。仅支持二分类和回归任务，不支持多分类。

## 项目结构

```
OpenMLStudio/
├── backend/analysis/              # 核心 ML 逻辑
│   ├── model_training.py          # 训练引擎
│   ├── model_prediction.py        # 预测引擎
│   ├── shap_analysis.py           # SHAP 解释
│   ├── random_forest_trainer.py   # 随机森林
│   ├── decision_tree_trainer.py   # 决策树
│   ├── xgboost_trainer.py         # XGBoost
│   └── linear_regression_trainer.py # 线性回归
├── frontend/                      # Streamlit Web UI
└── data/ml_models/                # 模型存储
```

## 适用场景

快速原型开发、教学演示、团队协作、可解释性研究。
