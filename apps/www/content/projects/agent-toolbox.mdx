> AI Agent 开发工具导航站，收录和分类构建 AI Agent 所需的各类工具和资源。涵盖向量数据库、文档解析、模型监控、网络爬虫、搜索 API、知识图谱等核心领域。支持用户创建和分享工具集合，每周持续更新。

开发 AI Agent 时，需要的工具散落在各处：向量数据库在一个网站，文档解析库在另一个，模型监控工具又在别处。找齐一套完整的工具链要花不少时间。

Agent Toolbox 把 AI Agent 开发所需的工具集中收录，按功能分类整理。

## 按开发环节分类

工具分为几个核心领域：向量数据库（Pinecone、Weaviate、Qdrant）、文档解析（用于 RAG 的 PDF/Word 处理）、模型监控（LangSmith、Weights & Biases）、网络爬虫（Firecrawl、Jina Reader）、搜索 API（Tavily、Perplexity）、知识图谱构建等。

每个工具都有简介、使用场景、定价信息和官网链接。通过标签系统可以快速筛选，比如"免费"、"开源"、"支持本地部署"等。

## 用户贡献的工具集合

除了浏览全部工具，用户可以创建自己的 Collection（工具集合）。比如"我的 RAG 技术栈"，包含选定的向量数据库、嵌入模型、文档解析器。这些集合可以公开分享，其他开发者可以参考。

每周会更新新工具，保持内容的时效性。导航站本身也是一个开发者社区，通过分享和讨论帮助彼此发现更好的工具组合。
