一个模仿经典 macOS 风格的交互式个人作品集网站，提供真实的操作系统体验。内置文本编辑器、终端、聊天室、iPod 音乐播放器等多个应用程序，支持 macOS、Windows 98、Windows XP 三种主题切换。

基于开源项目 [ryOS](https://github.com/ryokun6/ryos) 定制开发，使用 React + TypeScript + Vite 构建，支持移动端、平板和桌面全平台。
