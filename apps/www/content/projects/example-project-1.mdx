## Overview

This project is a personal blog and knowledge base template transformed from Animate UI. It combines beautiful animations with a powerful documentation system to create an engaging personal website.

## Features

- **Animated UI Components**: Leveraging Motion for smooth, delightful animations
- **MDX Support**: Write content in Markdown with React components
- **Dark Mode**: Beautiful dark and light themes
- **Responsive Design**: Perfect on all devices
- **Blog System**: Full-featured blog with tags and dates
- **Projects Showcase**: Dedicated page for displaying your work
- **Knowledge Base**: Organized notes and documentation

## Technical Highlights

### Architecture

Built on top of Fumadocs, this template uses Next.js App Router for optimal performance and SEO. The entire site is statically generated at build time.

### Design System

Follows a consistent design language with:

- Unified color palette (neutral grays with primary accent)
- Consistent spacing (Tailwind's spacing scale)
- Smooth animations (Motion-powered transitions)
- Accessible components

### Content Management

All content is managed through MDX files, making it easy to:

- Write in Markdown
- Embed React components
- Add custom styling
- Organize hierarchically

## Getting Started

1. Clone the repository
2. Install dependencies with `pnpm install`
3. Run the development server with `pnpm dev`
4. Start customizing!
