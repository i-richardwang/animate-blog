> 网页完整归档工具，一键捕获截图、元数据和全文本内容。支持多分辨率截图、明暗主题切换、动态内容处理和自定义样式注入。将网页的所有关键信息整合到一次操作中，让内容归档变得简单完整。

做竞品分析、设计参考或内容研究时，往往需要保存网页的视觉效果、页面标题描述、以及可搜索的文本内容。传统截图工具只提供图片，浏览器保存功能在面对动态加载和 JavaScript 渲染时容易出现问题。如果分别使用多个工具来获取这些信息，又需要手动维护它们之间的关联关系。

SiteSnapper 在一次操作中同时获取截图、元数据和文本内容，保持信息的完整性和一致性。

## 功能演示

### 1. 快速开始

![SiteSnapper 主界面](/projects/sitesnapper/main-interface.png)

在主界面输入要捕获的网页 URL。系统提供灵活的配置选项：

- **分辨率选择**：从 HD（1280×720）到 5K（5120×2880）五种清晰度
- **宽高比设置**：16:9、16:10、1:1 和 Full Page 完整页面模式
- **颜色主题**：支持明暗两种主题模式切换，适配现代网站的主题系统

点击"Snap Website"按钮，系统开始并行处理截图生成、元数据提取和文本内容获取，平均 2 秒内返回完整结果。

### 2. 捕获结果展示

![捕获结果页面](/projects/sitesnapper/capture-results.png)

捕获完成后，结果按信息类型分类展示。顶部显示网站图标和标题，支持一键下载或复制。网页描述清晰展示在独立卡片中，方便快速了解页面核心内容。

高清截图呈现在下方，显示实际输出分辨率信息，支持 PNG 或 JPEG 格式下载。每个卡片都配有对应的操作按钮，点击后会显示明确的成功反馈。

### 3. 完整文本内容

![文本内容输出](/projects/sitesnapper/content-output.png)

最下方展示提取的完整文本内容，经过智能清洗过滤掉脚本、样式等非内容元素，保留页面所有可见文字信息。

文本内容可一键复制，适合用于内容分析、搜索索引建立、NLP 处理等场景。相比传统截图工具只提供不可编辑的图片，文本内容的提取让归档的网页信息可以被二次处理和检索。

## 核心特性

**完整信息捕获**  
一次操作同时获取高清截图、页面标题和描述、品牌图标、完整文本内容。所有信息来自同一时刻的网页状态，保持彼此关联和一致性。相比分别使用截图工具、手动复制标题、提取文本等多个步骤，减少了操作复杂度。

**多分辨率支持**  
提供 HD、Full HD、QHD、4K、5K 五种分辨率选项，配合 16:9、16:10、1:1、Full Page 四种宽高比。通过组合不同参数，可以模拟网页在桌面端、平板、手机等不同设备上的显示效果。Full Page 模式自动捕获整个页面从头到尾，无需手动滚动拼接。

**主题模式切换**  
支持在明暗两种颜色主题下分别捕获网页。完美适配实现了主题切换的现代网站，让用户可以获取两种视觉风格的截图。对于设计师收集 UI 参考和进行竞品分析特别有用。

**动态内容处理**  
提供等待时间设置，确保异步加载的内容、动画效果完全显示后再截图。支持通过 CSS 选择器点击指定元素，可以关闭弹窗、展开菜单、触发特定交互状态。自定义 CSS 注入功能允许隐藏广告、调整布局或突出显示特定区域。

**智能元数据提取**  
自动识别并提取网页的标题、描述和品牌图标。这些元数据适合用于快速建立内容索引、生成引用信息或组织归档文件。相比手动复制，自动提取确保信息的准确性和完整性。

**文本内容获取**  
提取网页的所有可见文字内容，自动过滤脚本、样式等非内容元素。保持文本结构和可读性，适合用于离线搜索、内容对比、NLP 分析等场景。让归档的网页不仅能看，还能搜索和处理。

## 适用场景

适合设计师收集 UI 参考和竞品分析，通过不同分辨率和主题模式观察网页的响应式设计和视觉呈现。内容研究员可以建立完整的网页资料库，保留原网页变更前的状态。产品经理进行竞品追踪和功能对比时，可以快速获取视觉和文本记录。开发者用于视觉回归测试和 bug 记录，通过高级选项精准控制捕获时机。

将截图、元数据、文本内容整合到一次操作中，简化网页归档流程。
