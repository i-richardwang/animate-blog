EDA is the process of analyzing datasets to summarize their main characteristics using statistical graphics and other visualization methods.

## Key Steps

### Data Overview

Understanding the structure, types, and basic statistics of your data.

### Distribution Analysis

Examining the distribution of variables using histograms and density plots.

### Correlation Analysis

Identifying relationships between variables.

### Outlier Detection

Finding and handling unusual observations.

## Tools

- Pandas
- NumPy
- Matplotlib/Seaborn
